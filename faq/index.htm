FAQ

How to install the theme?

Installation instructions can be found from each theme's Github Readme page. There are 2 methods to install your theme.

Install via Git submodules

The easiest way is to install using Git. Assuming you already install Octopress with Git, you can add the theme to a sub folder on your blog's repository using Git submodules.

Install via downloaded package

Each theme is neatly packaged into a zip archive. You can download and copy it to your blog's repository.

Detailed instructions can be found on the theme's Github page, or on the README file in the package.

How to obtain updates for the theme?

If you installed using Git submodules, updating is simply issuing a git pull from the theme repository.

If you installed using the downloaded package, you would have to download a new package and overwrite the theme files manually.

How to uninstall the theme?

If you installed using Git submodules, follow the instructions from the theme's Github page. You have to remove the submodule entries, remove the theme from Git and then remove the files.

If you installed using the downloaded package, just remove the theme files manually.

Are you the official theme and plugins site for Octopress?

No we aren't. We are a third party site not associated with the official Octopress project.
